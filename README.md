# Sudoku

################################################################################################

Bu proje, hem web tarayıcısı üzerinden hem de .exe (masaüstü uygulaması) olarak çalışabilen bir Sudoku uygulamasıdır.

Web sürümü: Tarayıcı üzerinden oynanabilir

Masaüstü sürümü (.exe): Kurulum gerektirmeden çalışır

Doğru girilen:

Hücreler yeşil renkle işaretlenir

Satırlar ve sütunlar otomatik olarak kontrol edilir

Tüm sudoku doğru çözüldüğünde “Sudokuyu çözdünüz” bildirimi gösterilir

Bu proje, aynı mantığın hem web hem de masaüstü ortamında nasıl uygulanabileceğini göstermek ve kullanıcıya platformdan bağımsız bir Sudoku deneyimi sunmak amacıyla geliştirilmiştir.
